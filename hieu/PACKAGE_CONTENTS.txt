After running the script, these directories will be created:

.venv/                       | Python virtual environment
                             | Contains: Python executable, packages, etc.
                             | Size: ~1-2 GB (includes torch, transformers)

                             |

                             | Contains: FAISS index + embeddings
                             | Size: ~500 MB (depends on document size)
                             | ✅ Keep between runs for faster initialization
                             |

                             | Contains: Previously computed answers (JSON)
                             | Size: ~10-100 MB (grows with answers)
                             | ✅ Keep between runs for instant answers
                             |
uploads/                     | Uploaded documents
                             | Contains: Files teacher uploaded
                             | Size: depends on documents
                             |
__pycache__/                 | Python bytecode cache
                             | Auto-generated, safe to delete
                             | Size: negligible
                             |
server.log                   | Server execution log
                             | Contains: all server output & errors
                             | Size: ~100 KB per run
                             | Useful for debugging

🔄 WORKFLOW (What happens when you run run.sh)
================================================================================

1. Check uv installed
   └─ If not: pip install uv

2. Create .venv
   └─ uv venv .venv

3. Install dependencies
   └─ uv pip install -r requirements.txt

4. Download embedding model
   └─ python download_model.py
   └─ Downloads: keepitreal/vietnamese-sbert (~500 MB)

5. Start server
   └─ python main.py
   └─ Starts FastAPI on http://127.0.0.1:5004
   └─ Runs in background

6. Register with teacher
   └─ python client.py register
   └─ Tells teacher where our server is

7. Run evaluation
   └─ python client.py evaluate
   └─ Teacher sends 100 questions × 5 times
   └─ We answer all, store in cache
   └─ Takes 10-15 minutes

8. Get results
   └─ python client.py result
   └─ Fetch and display score

9. Cleanup
   └─ Kill server process
