================================================================================
SUPPORT & TROUBLESHOOTING
================================================================================

If run.sh fails:

   1. Check your Python version: python --version
   2. Check .env configuration
   3. Read SETUP.md (Troubleshooting section)
   4. Check server.log for errors
   5. Ensure network connectivity

Common issues & solutions:
   • "Port 5004 already in use" → Kill other process on that port
   • "Teacher URL not found" → Check .env TEACHER_BASE_URL
   • "Timeout waiting for server" → Check server.log
   • "Connection refused" → Check network & firewall
